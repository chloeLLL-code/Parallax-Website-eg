視差捲動效果

作者：Vineet Kumar

什麼是視差捲動？

視差捲動是一種網站設計技巧，讓頁面背景相較於前景以較慢的速度移動。當訪客往下滑時，就會呈現出 3D 深度，營造更具沉浸感的瀏覽體驗。視差源自於視覺錯覺：人眼會把靠近的物體看得比遠處的物體大，也會感覺遠處的物體移動得更慢。這項錯覺早已被運用在各種媒介中，以創造更逼真的效果。它最早出現在傳統動畫中，可追溯到迪士尼《白雪公主與七矮人》，也被像是《超級瑪利歐》等電玩採用。隨著 CSS 與 HTML 的進步，視差效果才演化為今日熟悉的網頁設計方式。

其中一個早期的瀏覽器視差捲動實作，是由網頁開發者 Glutnix 在 2007 年的部落格文章中分享，包含示範 JavaScript 與 CSS 2 的程式碼，當時甚至支援 Internet Explorer 6 與其他同時期瀏覽器。不過直到 2011 年，隨著 HTML5 與 CSS3 的問世，這項技術才在網頁設計師間流行起來。支持者認為，這是擁抱網路流動性的簡單方式；使用者擁護視差背景，讓網站能更吸引用戶並提升整體體驗。然而普渡大學 2013 年的研究指出：「……雖然視差捲動改善了使用者體驗的某些面向，卻不一定能提升整體使用者體驗。」


為什麼要使用視差圖形？

    設計師應該嘗試視差圖形的兩個主要理由如下：

  • 有助於講故事

    視差圖形非常適合用來說故事。好的設計就是溝通：當我們設計網站時，希望和訪客互動，向他們傳遞關於產品、服務或品牌的精彩敘事。視差捲動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓使用者不必跳轉頁面，就能看完整個品牌或產品的故事。

  • 提升使用者參與度

    視差圖形也能提升使用者參與度。降低網站跳出率是網頁設計的重要目標之一；跳出率代表訪客只看了一頁就返回或關閉分頁的比例。跳出率太高會傷害搜尋排名，因此任何能降低跳出率的作法都值得嘗試。一個精心設計的視差頁面能帶來更具動態感的體驗，讓使用者停留更久。


如果想在網站加入視差捲動效果，請留意以下建議。

  • 選擇性使用

    視差捲動聚焦在少數元素時效果最好，而不是每個頁面、每個區塊都使用。可以放在頁首或標題，或只用在首頁。視差應該強化網站設計，而不是分散使用者注意力。

  • 壓縮圖片

    視差效果會運用多種媒體檔案與 CSS 造型來營造深度。避免使用載入過慢的大型圖片，以免犧牲網站效能與使用者體驗。

  • 善用色彩

    視差不只關於前景與背景的圖像，也與色彩有關。背景可以使用較低飽和度的色調，前景則搭配鮮明色彩，能進一步加強深度感。
